# key-value

A keyboard event visualizer that displays key values, codes, and modifier states.

## Features

- [x] Display panel showing key value and key code on every keypress
- [x] Indicators for Alt, Ctrl, Meta, and Shift keys (True/False)
- [x] **Bonus:** Unique tone played on each keypress (frequency varies by key)
- [x] **Bonus:** Background color flash on keypress
- [x] Scrollable key history log

## Tech Stack

- HTML
- CSS
- JavaScript (vanilla)

## How to run

Open `index.html` in your browser — no build tools needed.
